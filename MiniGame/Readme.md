# 미니게임 데브 로그

# 1주차 수요일(23.09.13)


과제: 아이디어 추가

## 기획
#### 1. 재화 추가
  - 재화별 가치(스코어 점수 다르게)
    - 골드: 1점
    - 다이아몬드: 5점
  - 자전하면서 이동

#### 2. 속도 개선
  - 플레이어가 빨라지던거 없애기
  - 오브젝트의 다가오는 속도를 높이기

> 레퍼런스: 쿠키런-오븐브레이크 피버타임, 검은사막 모바일 다운로드 미니게임

### 결과
1-a. 재화 추가 (성공)
1-b. 자전(실패) 이동(성공) → 자전하면서 이동시키면 공전을 하는 문제 발생. 3d 좌표 문제인듯
2-a. 플레이어 속도 삭제 (성공)
2-b. 오브젝트 속도 증가 (실패: 시간부족)
